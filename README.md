# Compile Lab

本仓库由原编译原理实验仓库迁移整理而来，包含编译器源码、实验报告、课程文档和测试用例。

## 目录结构

- `src/`: 编译器项目根目录，由原 `cp-3230104703` 目录重命名而来。
- `src/src/`: 编译器核心 C++ 源码，包括 AST、语义分析、IR、优化和 RISC-V 代码生成。
- `src/reports/`: 各次实验报告、PDF 和配图资源。
- `src/appends/`: 附加测试用例和 bonus 相关材料。
- `src/sp26-tests/`: 编译器项目内使用的测试套件。
- `sp26/`: 课程文档站点源码。
- `sp26-tests/`: 独立的课程测试套件。

## 构建

在编译器项目目录下构建：

```bash
cd src
make
```

构建完成后会在 `src/` 下生成 `compiler` 可执行文件。生成的 parser、lexer、目标文件、依赖文件和可执行文件不会提交到 Git。

## 测试

在编译器项目目录下运行指定实验的测试：

```bash
cd src
python3 sp26-tests/test.py lab4 .
```

需要测试其他实验时，将 `lab4` 替换为对应实验名。
